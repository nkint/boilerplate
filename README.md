Install Atom from atom.io, with the following additional packages: 

  language-glsl
  linter
  linter-eslint
  linter-ui-default
  minmap
  file-icons
  fonts
  indent-guide-improved
  highlight-selected

Then clone the project and run:

  git clone https://github.com/nkint/boilerplate
  cd bolierplate
  npm install yarn
  yarn
  yarn start
